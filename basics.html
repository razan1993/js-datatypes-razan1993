Javascript

Javascript is een functional programming language. Met javascript kun je zowel client-side als server-side rendering doen. Client-side houdt in dat javascript op jouw browser uitgevoerd wordt en Server-side houdt in dat het op een extern computer uitgevoerd wordt. Lees je heel goed in wat je met javascript kunt doen: hier

Opdracht 4: Getallen kun je bijelkaar optellen, aftrekken, delen en restdeling.

  • Maak een variabel "getal1" aan met een waarde 10
  • Maak een variabel "getal2" aan met een waarde 20
  • Op het scherm wil ik het volgende zien:
    • getal1:10, getal2:20, operator:+, antwoord=30
    • getal1:10, getal2:20, operator:-, antwoord=10
    • getal1:10, getal2:20, operator:/, antwoord=2
    • getal1:10, getal2:20, operator:%, antwoord=0
LET OP: gebruik de waarde van "getal1" en "getal2" om bijelkaar op te tellen, af te trekken, te delen en restdeling te doen.
